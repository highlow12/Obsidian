printf: 표준 출력
scanf_s: 표준 보안 입력
```c
//출력
printf("%d", a);

//입력
scanf_s("%d",&a);
```
## 포맷
|출력 형식|설명|비고|
|----|----|----|
|%d|부호 있는 정수||
|%c|문자 하나||
|%f|실수|%와 f 사이에 수를 넣으면 소수점 n자리 수까지 출력 그 밑은 반올림|
|%lf|긴 실수(double)|float 에도 사용 가능|
|%s|문자열||


# string
문자열은 scan 할 때 변수 앞에 &을 붙이지 않는다
scanf_s에 최대 입력 가능 문자를 입력한다

```c
#include <string.h>


// char
scanf_s("%c",chr);

// string
char str[256];
scanf_s ("%s",str,256);
```
