# 5장 쉘
# 쉘?
- 쉘은 사용자와 운영체제 사이에 창구 역할을 하는 소프트웨어
- 명령어 처리기(command processor)
- 사용자로부터 명령어를 입력 받아 이를 처리
# 쉘의 종류
| 쉘의 종류 | 쉘 실행 파일   |
| ----- | --------- |
| 본 쉘   | /bin/sh   |
| 콘 쉘   | /bin/ksh  |
| C 쉘   | /bin/csh  |
| Bash  | /bin/bash |
| tcsh  | /bin/tcsh |
