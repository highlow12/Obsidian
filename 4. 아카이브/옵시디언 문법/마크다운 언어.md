# 제목
'#'은 제목을 나타낼 수있는 기호이다
중첩하면 작은 제목이 된다 '#'이후 띄어쓰기를 해야한다
# 제목
## 제목
### 제목
#### 제목

# 수평선
\*, -을 6개 사용한다
******
------
---

# 체크박스
'- [ ]'를 사용하여 표시한다
안에 x 를 넣으면 체크표시가 됨
- [ ] 
- [x] 
# 강조 및 취소선
## 이탤릭
*\* 두개*
_\_ 두개_

## 굵게
**\* 네개**

## 하이라이트
==\= 기호 네개==

## 취소선
~~물결표 네개~~

## 인용문
>\> 을 사용하면 인용할수 있다 
>> 인용문 안에서도 사용 가능하다
