# 상속이란
- **상속**<sub>inheritance</sub> 객체 지향 프로그래밍의 또다른 주요 요소
- 클래스를 선언 할 때 다른 클래스의 속성과 메소드를 상속 가능
- 상속을 받는 클래스: 자식/하위 클래스
- 상혹을 하는 클래스: 부모/상위 클래스
## 상속을 하는 클래스 선언
```python
class 자식클래스(부모클래스):
	클래스본문
```
