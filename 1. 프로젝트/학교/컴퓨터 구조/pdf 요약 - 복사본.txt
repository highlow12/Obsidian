### 컴퓨터 시스템 개요 상세 요약

---

#### 1장: 컴퓨터의 정의와 구조

- **컴퓨터란?**  
  데이터를 자동으로 처리하는 기계로, 프로그램이라고 하는 명령어 목록에 따라 주기억장치에서 명령을 불러와 실행한다.
  
- **컴퓨터 아키텍처(Architecture)**  
  - 프로그래머에게 보이는 시스템의 속성들로, 프로그램 실행에 직접적으로 영향을 미치는 요소들을 포함한다.  
  - **주요 속성**: 명령어 집합(Instruction set), 데이터 표현 방식, 입출력(I/O) 기법, 주소 지정 방식 등.
  
- **컴퓨터 조직(Organization)**  
  - 컴퓨터 아키텍처를 구현하는 물리적 구성 요소의 연결 및 동작을 의미하며, 하드웨어의 세부적인 특성을 설명한다.  
  - **주요 속성**: 제어 신호(Control signals), 메모리 기술, 컴퓨터와 주변 장치의 인터페이스.

#### 2장: 컴퓨터 하드웨어와 소프트웨어 구성

- **하드웨어(Hardware)**  
  - 정보의 전송 경로와 처리를 담당하는 물리적인 장치들로, 중앙처리장치(CPU), 기억장치, 입출력장치로 구성된다.
  
- **소프트웨어(Software)**  
  - 하드웨어의 동작을 제어하고, 명령을 통해 데이터의 처리 방향 및 시간을 지정한다.  
  - **종류**:
    - **시스템 소프트웨어**: 운영체제(OS) 등, 컴퓨터의 기본적인 동작을 제어.
    - **응용 소프트웨어**: 워드 프로세서, 웹 브라우저 등, 사용자 작업을 수행.

#### 3장: 컴퓨터의 주요 구성 요소

- **중앙처리장치(CPU)**  
  - 프로그램 실행과 데이터 처리의 중심 역할을 수행하며, 내부적으로 프로세서와 레지스터로 구성된다.
  
- **기억장치(Memory)**  
  - **주기억장치(Main Memory)**: CPU와 가까운 곳에 위치하며, 프로그램 실행에 필요한 데이터를 일시적으로 저장한다. 고속 액세스가 가능하지만 비휘발성은 아님.
  - **보조기억장치(Auxiliary Storage Device)**: 하드디스크, 플래시 메모리 등, 데이터의 영구 저장을 담당한다. 저장 밀도가 높고 비트당 비용이 저렴하지만, 속도가 느리다.

- **입출력장치(I/O Device)**  
  - 사용자와 컴퓨터 간의 상호작용을 담당하며, 데이터의 입력과 출력을 처리한다.
  - 예: 키보드, 마우스, 프린터, 네트워크 인터페이스 등.

#### 4장: 정보의 표현과 프로그램 언어

- **정보의 표현**:  
  - 컴퓨터는 2진수(0과 1)로 데이터를 표현하며, 프로그램 코드와 데이터를 저장 및 처리한다.

- **컴퓨터 언어**:  
  - **기계어(Machine Language)**: CPU가 직접 이해할 수 있는 이진 코드. CPU마다 명령어 집합이 다르다.
  - **어셈블리 언어(Assembly Language)**: 기계어와 1:1로 대응하는 저급 언어로, 어셈블러에 의해 기계어로 번역된다.
  - **고급 언어(High-Level Language)**: 사람이 이해하기 쉬운 영문과 숫자로 구성된 언어. 컴파일러를 통해 기계어로 변환된다(C, C++, PASCAL 등).

#### 5장: 프로그램 개념과 시스템 버스

- **프로그램 개념**  
  - **하드웨어 프로그래밍**: 특정 연산을 수행하기 위해 하드웨어를 재구성하는 방식(유연성 부족).
  - **소프트웨어 프로그래밍**: 일반적인 하드웨어에서 새로운 명령어 세트를 제공하여 다양한 작업을 수행할 수 있도록 제어 신호를 생성.

- **시스템 버스**  
  - 컴퓨터의 주요 구성 요소(CPU, 메모리, I/O 장치) 간의 데이터를 주고받는 통신 경로.
  - **주소 버스(Address Bus)**: CPU가 특정 기억장치 주소로 데이터를 전송하는데 사용.
  - **데이터 버스(Data Bus)**: 데이터와 명령어를 전송하는 데 사용.
  - **제어 버스(Control Bus)**: 메모리 읽기/쓰기, 인터럽트 요청, 클럭 신호 등 시스템의 동작을 제어.

#### 6장: 컴퓨터 시스템의 발전과 역사

- **폰 노이만 구조**  
  - **Stored-program 개념**: 프로그램과 데이터를 기억장치에 저장하고 CPU가 이를 해석하여 실행.
  - **주요 구성요소**: 프로그램 제어 유닛, 산술논리연산장치(ALU), 주기억장치, 입출력장치.

- **반도체 기술 발전**  
  - **트랜지스터**: 진공관을 대체하여 컴퓨터의 크기를 줄이고 속도를 높임.
  - **집적 회로(IC)**: 수천 개의 트랜지스터를 하나의 칩에 집적, 컴퓨터 성능 향상.
  - **VLSI**: 수십만 개 이상의 트랜지스터가 포함된 초대규모 집적 회로로, 개인용 컴퓨터(PC)와 고성능 컴퓨터의 발전을 이끈 주요 기술.

#### 7장: 컴퓨터 시스템의 분류

- **개인용 컴퓨터(PC)**  
  - 소형화 및 저가화된 컴퓨터로, 고속 I/O와 멀티미디어 기능을 제공. 다양한 종류(데스크탑, 노트북, 태블릿 등)로 발전.
  
- **중형 컴퓨터**  
  - **워크스테이션**: 고속 그래픽 처리와 시뮬레이션에 사용.
  - **슈퍼미니컴퓨터**: 다중 프로세서 구조로 서버 환경에서 사용.

- **메인프레임(Mainframe)**  
  - 대규모 데이터 처리와 보안, 통신 기능이 강화된 컴퓨터로, IBM zEnterprise 계열 등이 대표적.

- **슈퍼컴퓨터(Supercomputer)**  
  - 고속 계산과 대용량 데이터 처리에 특화된 컴퓨터로, VLSI 설계, 항공우주공학, 기상 예보 등의 분야에서 활용.

- **임베디드 컴퓨터(Embedded Computer)**  
  - 특정 장치의 내부에 포함되어 그 장치의 동작을 제어하는 컴퓨터. 실시간 처리 능력이 중요하며, 가전제품 및 이동기기 등에 사용된다.

#### 8장: 컴퓨터 아키텍처와 성능 향상

- **무어의 법칙**  
  - 칩의 트랜지스터 수가 18개월마다 두 배로 증가하여 컴퓨터 성능이 지속적으로 향상됨.
  
- **황의 법칙**  
  - 메모리 반도체의 집적도가 매년 두 배로 증가하는 이론으로, 고성능 메모리 반도체의 발전을 설명.
